## Badge

### Attributes

| name    | description                              | type                                                                                   | default |
| ------- | ---------------------------------------- | -------------------------------------------------------------------------------------- | ------- |
| type    | color type                               | neutral, primary, secondary, accent, info, success, warning, error, ghost, link, glass | neutral |
| size    | size                                     | ISize                                                                                  | md      |
| outline | transparent badge with [colorful] border | boolean                                                                                | false   |
