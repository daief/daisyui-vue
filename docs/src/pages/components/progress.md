## Progress

### Attributes

| name  | description            | type                                                               | default |
| ----- | ---------------------- | ------------------------------------------------------------------ | ------- |
| type  | progress color type    | netural, primary, secondary, accent, info, success, warning, error | netural |
| value | current progress value | number                                                             | 0       |
| max   | max progress value     | number                                                             | 100     |
